Project developed at the [RocketSeat GoStack Bootcamp](https://rocketseat.com.br) using Node.js, for a barber scheduling app.

## Development Using

- Node.js
- Express
- nodemon
- Sucrase
- Docker
- Sequelize
- PostgreSQL
- node-postgres
- Redis
- MongoDB
- Mongoose
- JWT
- VS Code with ESLint
